# 📚 Documentación del Proyecto BGnius App

Esta carpeta contiene toda la documentación técnica y funcional del proyecto.

## 📂 Estructura de Directorios

### 📝 Documentos Procesados (Markdown)
En la raíz de esta carpeta encontrarás los documentos convertidos a formato Markdown para facilitar su lectura, indexación y procesamiento por herramientas de IA.

- **`screen_specifications.md`**: Detalle completo de las especificaciones de cada pantalla (basado en *VITA Basic.pdf* y *DISEÑO APP 3.0.pdf*).
- **`navigation.md`**: Diagrama de flujo y arquitectura de navegación de la aplicación.
- **`navigation_analysis_report.md`**: Reporte de auditoría comparando la implementación real (`app_router.dart`) con la documentación.
- **`MQBIS - DM01 - Requerimientos de proyecto - PR1 - BGnius.md`**: Versión markdown del documento oficial de requerimientos.
- **`ux_accessibility_audit.md`**: Auditoría de Experiencia de Usuario y Accesibilidad.
- **`apk_build_guide.md`**: Guía para generar el APK de la aplicación.
- **`auth_module_technical_spec.md`**: Especificación técnica del módulo de autenticación.

### 🗄️ carpetas
- **`features/`**: Documentación específica por feature.
- **`pantallas/`**: Documentación visual o específica de pantallas individuales.
- **`componentes/`**: Documentación de componentes reutilizables.

### 📦 `originales/`
Esta carpeta contiene los archivos fuente originales que no están en formato Markdown. **Úsalos solo como referencia de respaldo.**

- PDFs de Diseño (*VITA Basic.pdf*, *DISEÑO APP 3.0.pdf*, *MEDIDAS APP V3.pdf*)
- Documentos Word (*MQBIS...docx*)
- Recursos gráficos originales (Logos, Iconos)

---
*Nota: Para cualquier trabajo de desarrollo o consulta, por favor referirse primariamente a los archivos `.md` en la raíz de `docs/`.*

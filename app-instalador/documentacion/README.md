## 🤖 Reglas para Asistentes de IA

> **IMPORTANTE**: Al trabajar con documentación en este proyecto:
> - ✅ **USAR**: Archivos `.md` para análisis, búsqueda y referencia
> - ❌ **NO USAR**: Archivos `.docx` o `.pdf` (son respaldos originales)
> - Los archivos Markdown son la fuente de verdad actualizada
> - Los archivos originales se mantienen únicamente como respaldo histórico

## 📝 Convenciones

- Todos los documentos importantes deben tener una versión `.md`
- Los diagramas se implementan usando Mermaid cuando sea posible
- Mantener los archivos `.md` sincronizados con cualquier cambio en los originales

## 🔄 Actualización de Documentación

Si necesitas actualizar la documentación:

1. Edita el archivo `.md` correspondiente
2. Documenta los cambios en el commit de Git
3. No modifiques los archivos `.docx` o `.pdf` originales (son históricos)
